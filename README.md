# Hangman Game – CodAlpha Internship Task 1

##  Project Description
This is a simple text-based Hangman Game created for *CodAlpha Internship – Task 1*.  
The game selects a random word from a predefined list and the player must guess the word one letter at a time.

##  Concepts Used
- Python random module  
- while loop  
- if-else conditions  
- Lists & strings  
- User input handling  

##  Features
- 5 predefined words  
- 6 incorrect attempts allowed  
- Real-time updating word display  
- Duplicate guess checking
